# How to use this curriculum

## Paths

| Reader | Suggested path |
|--------|----------------|
| Busy clinical fellow | Decision-focused reading → absolute effects → trial appraisal → journal-club checklist chapters |
| Research-oriented trainee | Estimands → DAGs/bias → observational designs → prediction models → transportability |
| Journal club lead | Pick one paper archetype; use the matching chapter’s framework live |

## Chapter pattern

Each chapter aims for:

1. Learning objectives  
2. Conceptual core  
3. Frameworks / instruments (described, not copyrighted dumps)  
4. Mathematics with defined symbols  
5. Worked example (synthetic data)  
6. Pitfalls  
7. Clinical and epidemiologic notes  
8. Summary and practice  

## Web vs print habits

- Use search for terms (NNT, collider, TRIPOD).  
- Math renders with MathJax.  
- Figures enlarge in a new tab if needed.  
- Report errors via GitHub Issues on this repo.
